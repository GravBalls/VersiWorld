Production Brief

Working Title

    VersiWorld

Game Objective

    Get a ball into a goal.

Player Role

    Player will be given simple controls that they can use to manipulate the world that the ball is in, these controls will vary per level (and will each be clearly marked). The player will have an unobstructed side view of the entire level as the game will be more of a puzzle.

Physics Mechanics

    The objective ball will be a simple physics based ball traveling through an environment that will often contain other physics objects. The player's control of the world will be include changing an objects properties (friction, bounciness, etc.) or activating objects like moving platforms, as well as changing the entire world's properties like gravity.

General

    The game will be set up as a puzzle and the "fun" of the game is from discovering how each control works and using a and overcoming the limited control set to solve the puzzles.

    Our plan for implementation will start with control so we can standardize how objects in the world respond to the player's input, then we will move onto implementing these objects before moving to the design of the levels.
